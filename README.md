# Chat AI Moderator — Fabric 1.21.11

Клиентский мод для Minecraft Fabric 1.21.11. Он читает входящие сообщения чата, ищет оскорбления/подозрительные сообщения и выводит большое предупреждение на экран.

Важно: это локальный «мини-ИИ» без внешнего API. Он работает офлайн: нормализует текст, ловит слова из конфига и простые обходы через символы/пробелы.

## Что умеет

- Проверяет обычный чат игроков.
- Проверяет системные сообщения сервера, потому что на некоторых серверах чат приходит не как `CHAT`, а как `GAME`.
- Ловит простые обходы: пробелы, символы, повтор букв.
- Показывает огромный красный текст: `НАРУШЕНИЕ В ЧАТЕ`.
- Создает конфиг `config/chat-ai-moderator.json`.

## Как собрать через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи туда все файлы из этого архива.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build Fabric Mod**.
5. После сборки скачай artifact с `.jar`.
6. Положи `.jar` в папку `.minecraft/mods` вместе с Fabric API.

## Как собрать на ПК

Нужно:

- Java 21.
- Gradle 9.6+.

Команда:

```bash
gradle build
```

Готовый `.jar` будет здесь:

```text
build/libs/chat-ai-moderator-fabric-1.0.0.jar
```

На Windows можно попробовать `build.bat`.

## Как добавить свои слова

После первого запуска появится файл:

```text
.minecraft/config/chat-ai-moderator.json
```

Пример:

```json
{
  "enabled": true,
  "overlayDurationMs": 6500,
  "minToxicityScore": 0.65,
  "scanSystemMessages": true,
  "playSound": true,
  "insultWords": [
    "слово1",
    "слово2",
    "слово3"
  ],
  "detectObfuscatedWords": true
}
```

После изменения конфига перезапусти Minecraft.
